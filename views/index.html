Personal Library

Description

Keep track of your books, no matter how large your home library has become!

Features of this Personal Library

  • add book
  • add comment
  • delete book(s)
  • retrieve book(s)

Add books, comments, delete books

Select a book to see it's details and comments

    Retrieve books

    You can retrieve a list of all books or just information about a single book.

    API call

    get all books

    /api/books

    get specific book

    /api/books/_id
